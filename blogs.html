Discuss the scope of var, let, and const

A variable declared with var is a function scope if declare inside of a function otherwise global scope. And a variable declared with let and const is block scope, which mean if we declare a variable with let and const inside a curly braces {} (block of code) the variable will only accessed inside the block.

Tell us the use cases of null and undefined

Null is an assignment value. It used to indicate that a variable is supposed to have no value. It's different from being empty or having zero.We can use it to assign a variable which has no value and is empty. And Undefined means we create a variable without assign a value and if we want to access a variable which is not declared then we will also get an undefined value

What do you mean by REST API?

REST API is a set of guidelines that allow application to communicate with each other over the internet. REST API is one of the most popular type of API architecture. We can communicate with servers using protocol like HTTP. REST stands for representational State Transfer. REST API's are state less so we can use it in the other other platform like Android or desktop application.
